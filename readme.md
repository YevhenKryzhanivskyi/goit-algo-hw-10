               Обчислення визначеного інтеграла методом Монте-Карло


## Результати
1.1 Метод Монте-Карло: 2.6676 (для N = 1 000 000)
1.2 Quad (SciPy): 2.6667
1.3 Аналітичний розрахунок: 2.6667


---

## Висновки
1.1 Метод Монте-Карло дає результат, дуже близький до аналітичного.
1.2 Похибка зменшується зі збільшенням кількості випадкових точок.
1.3 Для простих функцій аналітичний метод завжди точніший, але метод Монте-Карло є ефективним чисельним способом для складних інтегралів, де аналітичне рішення знайти важко.
1.4 Використання функції `quad` підтверджує правильність розрахунків.
